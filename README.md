# godgame
### 作成したものを保存する
追加した全てのファイルを上げる
~~~
git add -A
~~~

### 作成したものをブランチに乗せる
~~~
git commit -m "なにを追加したか、変更したかを書く"
~~~

### commitしたものをoriginに挙げる
~~~
git push
~~~

### Originの状態に自分も状態も合わせる
誰かが何か追加してそれを使いたい＆pushする前にやって
~~~
git pull
~~~

### ブランチを見る
~~~
git branch
~~~

### ブランチを切り替える
~~~
git checkout ブランチ名
~~~

ブランチを作成してそこに移動する
~~~
git checkout -b ブランチ名
~~~

### いまの状態を見る
~~~
git status
~~~

### コミット履歴を見る
~~~
git log
~~~
詳しく見る（qで戻る）
~~~
tig
~~~
